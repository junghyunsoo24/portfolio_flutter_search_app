# 검색엔진 앱

## 활동
* 24년 겨울방학때 자연어처리 관련 주제인 검색엔진 프로젝트 진행

## 내가 한 구현기술
* 플러터를 통한 웹/앱 프론트 진행
* 웹/앱 카카오 로그인 진행
* ui/ux
* flask server랑 http통신(검색 텍스트 등록 기능, SBert/Cos로 검색 기능)
